# Getting started with DRIVEN PASS

## `Backend settings:`

### Run

 - *npm install*
 - *npm prisma generate dev*
 - *npm run dev*
 
## `.env SETUP`

### DATABASE_URL
 - postgres://user:password@host:port/database_name

### PORT=port number
 - Default 5000

### CRYPTR=key
 - Your super secret key

### BCRYPT=salts
 - Number of salts, e.g. 10

### JWT_KEY=key
 - One more super secret key

### MODE=DEV

#

## `Frontend settings:`

### Run

 - *npm install*
 - *npm prisma generate dev*
 - *npm run start*
